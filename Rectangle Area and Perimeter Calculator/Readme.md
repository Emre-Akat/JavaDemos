# Rectangle Area and Perimeter Calculator

## Description
This is a Java program that calculates both the area and perimeter of a rectangle. It takes the length and width of the rectangle as input from the user, calculates the area using the formula `length * width` and the perimeter using the formula `2 * (length + width)`, and prints the results to the terminal.

## Features
- The project was developed using IntelliJ IDEA.
- Simple user input to calculate both area and perimeter of a rectangle.

# Dikdörtgen Alan ve Çevre Hesaplayıcı

## Açıklama
Bu program, dikdörtgenin alanını ve çevresini hesaplayan bir Java programıdır. Girdi olarak dikdörtgenin uzunluğunu ve genişliğini kullanıcıdan alır. Alanı `uzunluk * genişlik` formülü ile, çevresini ise `2 * (uzunluk + genişlik)` formülü ile hesaplar ve sonuçları terminale yazdırır.

## Özellikler
- Proje için IntelliJ IDEA kullanılmıştır.
- Basit kullanıcı girdisi ile dikdörtgenin alanı ve çevresi hesaplanabilir.
